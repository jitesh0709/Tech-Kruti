## 📌 Project Overview

**Techkruti** is a Department of CSE (Data Science) coding event where participants solve real-world and AI-powered problem statements using modern development tools. This repository contains the source code developed during the **Vibe Coding** competition.

---

## ✨ Features

* 🤖 AI-powered functionalities
* 🎨 Modern responsive UI
* ⚡ Fast performance with Vite
* 📱 Mobile-friendly design
* 🔒 Clean and scalable architecture
